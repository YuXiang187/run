    U盘图标

    先在文件夹选项取消勾选“隐藏已经文件类型的扩展名”和“显示隐藏的文件”

    1. 首先制作好一个.ico格式的图片，然后保存在U盘根目录中，并将图标命名为favicon.ico
    2. U盘根目录新建一个文本文件并命名为autorun.inf
    3. 用记事本工具打开autorun.inf文件，然后在里面输入下面的内容并保存关闭（注意要区分两行写）[autorun]
      icon=favicon.ico
    4. 将autorun.inf和favicon.ico文件都设置成只读和隐藏
    5. 将U盘拔出然后插上电脑，OK

    卸载预装

    1. 下载安装Minimal_ADB_Fastboot_v1.4.3、ADB

    2. 手机连接电脑，打开调试模式，通知栏连接设置为管理文件

    3. 进入命令提示符，切换到ADB根目录，然后输入下面命令：

      adb devices → adb shell

    4. 查找包命令：pm list packages（查看包对应程序可以下一个查看包名APP）

    5. 卸载包命令：pm uninstall -k --user 0 <package>

    6. 强制停止vivo系统自动更新命令：pm disable-user com.bbk.updater

    文件共享

    1. 打开计算机管理→本地用户和组，在用户一栏右键点击新用户，该输入的输入，取消勾选下次登陆时须更改密码，勾选用户不能更改密码、密码永不过期，完成后关闭
    2. 找到共享文件夹，在共享一栏右键点击新建共享，把你需要共享的文件夹填进去，OK

    附：Windows7启动电脑时老是要先选择账户的解决办法

    1. 按下Win+R运行control userpasswords2命令
    2. 选择一个需要自动登录的账户，再取消勾选要使用本机、用户必须输入用户名和密码复选框，点击确定即可

    如何使用？

    打开Chrome（当然别的浏览器也行）后打开开发者工具，直接将代码复制到Console栏，按Enter键即可

    注意事项：

    • 可以后台运行，只要不关闭页面，代码就会一直运行
    • 签到成功后会在Console栏输出“XXX时间完成签到”等字样，如果你这里已经签上了老师那边显示没数据，那就是老师网络的问题，老师问你的时候直接说“我签了”就完事了
    • Console栏可以设置只输出Info以屏蔽错误信息
